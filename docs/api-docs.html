CycloneDX Generator (cdxgen) Server (latest)

Download OpenAPI specification:Download

URL: https://cyclonedx.slack.com/archives/C04NFFE1962 License: Apache 2.0

cdxgen is a CLI tool, library, REPL, and server to create a valid and compliant CycloneDX Bill of Materials (BOM) containing an aggregate of all project dependencies for C/C++, Node.js, PHP, Python, Ruby, Rust, Java, .Net, Dart, Haskell, Elixir, and Go projects in JSON format.

CycloneDX is a full-stack BOM specification that is easily created, human and machine-readable, and simple to parse.

query Parameters
type
string
Default: "universal"

Project Type

See supported project types
multiProject
boolean
requiredOnly
boolean

Include only the packages with required scope on the SBOM

noBabel
boolean

Do not use babel to perform usage analysis for JavaScript/TypeScript projects

installDeps
boolean
Default: true

Install dependencies automatically for some projects. Defaults to true but disabled for containers and oci scans

projectId
string

The UUID of the project. You must provide the UUID or the projectName and projectVersion (or all three)

projectName
string

Dependency Track project name. Default use the directory name

projectGroup
string

Dependency Track project group

projectVersion
string
Default: ""

Dependency Track project version

parentUUID
string

UUID of the parent Dependency Track project

serverUrl
string

URL to the Dependency Track API server

apiKey
string

API key for the Dependency Track API server

specVersion
string
Default: "1.5"

CycloneDX Specification version to use

filter
Array of strings

Filter components containing this word in purl. Multiple values allowed

only
Array of strings

Include components only containing this word in purl. Useful to generate BOM with first party components alone. Multiple values allowed

autoCompositions
boolean
Default: true

Automatically set compositions when the BOM was filtered. [boolean]

gitBranch
string

Git branch used when cloning the repository. If not specified will use the default branch assigned to the repository
